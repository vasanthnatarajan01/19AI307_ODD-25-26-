## Sourcecode.java:

```java
import java.util.Scanner;
public class Main
{
    public static void main(String[] args)
    {
        Scanner scan = new Scanner(System.in);
        String name = scan.nextLine();
        System.out.print("Hello, "+name);
    }
}
```
